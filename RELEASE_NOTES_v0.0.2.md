# Research OS v0.0.2

This release focuses on packaging and polish.

## Highlights

- Research creation simplified to default research flow
- Calendar items support start and end time
- Daily schedule view is clearer and closer to a timetable layout
- Desktop packaging is automated for macOS and Windows
- Release size is reduced with release profile optimizations

## Downloads

- macOS: `Research OS_0.0.2_aarch64.dmg`
- Windows: `Research-OS_v0.0.2_x64_windows.zip`

## Notes

- Windows build is a portable zip package. Extract and run.
- Data is stored locally in LocalStorage by default.
