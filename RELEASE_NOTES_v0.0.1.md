# Research OS v0.0.1

Research OS 的第一个可发布版本。

## Highlights

- Research 项目采用固定阶段流转：idea / research / writing / submission / revision
- 杂事支持 workflow checklist 形式管理
- 日历支持按日期查看事项，并与研究和杂事的截止日期联动
- 研究阶段详情支持文件、链接和阶段内容编辑
- 支持桌面端 Tauri 打包

## Downloads

- macOS: `Research OS_0.0.1_aarch64.dmg`
- Windows: `Research-OS_v0.0.1_x64_windows.zip`

## Notes

- Windows 版本为便携包，解压后即可运行
- 数据默认保存在本机 LocalStorage 中
